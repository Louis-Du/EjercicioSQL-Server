# Formulario

Este ejercicio simula la gestión de un formulario de datos en una base de datos relacional.

## Resumen
Se crean tablas para almacenar información básica y se practican operaciones de inserción, consulta y manipulación de datos.

## Ejemplo de consulta
```sql
SELECT * FROM Personas WHERE edad > 18;
```

## Resultado esperado
| id | nombre  | edad |
|----|---------|------|
| 1  | Andrea  | 22   |
| 2  | Carlos  | 25   |

## Archivos
- `Schema.sql`: Estructura de la base de datos
- `Data.sql`: Datos de ejemplo
- `Query.sql`: Consultas de práctica
- `ModeloRelacional.png`: Diagrama relacional
